DICraft

Convert DICOM files into voxel/pixelcloud, edit Minecraft style and 3D-print them.

This project tries to fill the gap between X-ray imaging (slices, single images, ultrasound images) and 3D-printing.
For medical and scientific use (you will find application cases).

Supported platforms

  • Linux is the main dev platform, because there is all we need.
  • MAC turned out to be the second choice, because the Unix like system makes it easy for everything needed to run.
  • Windows is the target platform, everyone has it and most 3D-printing software works better on it.

What we have achieved so far

  • Bunch of DCM-files (slices) in a "useful" format
  • DICraft editor mode
  • STL-exported jaw (in MeshLab)
  • jaw print in progress
  • 3D-printed jaw
  • Human Brain, scale is NOT 1:1!
